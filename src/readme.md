# Football Central - React Frontend Web Application

## Project Description
Football Central is a functional frontend web application built as Phase-2 of my web development project. It solves the real-world problem of providing football fans with a centralized hub for tracking match schedules, recent results, and reading the latest sports news. The project utilizes component-based architecture and frontend routing.

## Features
- **4 Distinct Pages:** Home, News, Matches, and Login.
- **Responsive Design:** Adapts smoothly to desktop and mobile devices.
- **Client-Side Routing:** Fast navigation without page reloads using React Router.

## Setup Instructions
1. Clone this repository to your local machine.
2. Ensure Node.js is installed.
3. Open terminal in the project directory and run `npm install`.
4. Run `npm run dev` to start the development server.
5. Open your browser and go to `http://localhost:5173`.
